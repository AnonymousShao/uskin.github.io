## Usage
```js
<InputSearch onClick={listener} width={300} type="light"/>
```

## API
|Name       |Type     |Default     |Descrition                    |
|-----------|---------|------------|------------------------------|
|onChange   |function |            |返回搜索框内的文字段                |
|width      |number   |            |搜索框的长度                     |
|type       |string   |            |搜索框的样式，有默认和light样式      |
